# Reference App (Operator Command Center)

This folder will hold the **facilitator's reference solution** App that demos what attendees will build by the end of the workshop. Build it ahead of time so:

1. You can show "here's the end state" at the welcome (1:00-1:10).
2. Attendees stuck on Module 5 integration can peek at a working example.

## What it includes

- AppKit shell with 3 tabs (Labor, Sales & Inventory, Sentiment) + an "Ask Genie" tab
- Embedded Genie space and AI/BI dashboard
- "Recommended Actions" sidebar driven by `ai_query()` and `<FMAPI_ENDPOINT>`
- LCE branding (logo, primary color, dark navbar)

## How to build

Use the same prompts in the Lab Companion Guide. Run them as the facilitator a few days before the workshop and check the app deploys cleanly.
